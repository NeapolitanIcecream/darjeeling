# 2026-06-10 L1 age factoid path

## Goal

Address the weak L1 coverage bottleneck with a conservative native Rust path for
a low-entropy, locally verifiable request family observed in cached smoke traces:
`how old is <person>`.

## Design

The new path is intentionally narrow:

- Accepts only utterances normalized as `how old is <person>`.
- Emits `qa_factoid` with a single `person` slot.
- Allows short proper-name-like token sequences.
- Rejects high-risk objects or determiners such as `the earth`, `my dog`, or
  `it`.

This is a hand-written baseline path, not a replacement for L1 coding-agent
evolution. Future broader L1 coverage should still go through the Rust
coding-agent harness and outer replay promotion.

## Validation

Focused checks:

```bash
cargo test --manifest-path native/l1_programbank/Cargo.toml
uv run pytest tests/test_replay_runtime.py \
  tests/test_compiler_loop.py::test_compiler_generation_promotes_l0_cache_without_gold_leakage -q
```

Result:

- Rust tests: 12 unit tests + 1 JSONL worker test passed.
- Python focused tests: 5 passed.

Cached replay smoke:

```bash
mkdir -p runs/l1-age-path-smoke-r1
cp runs/l2-list-fallback-tuned-3k-r1/teacher_cache.jsonl \
  runs/l1-age-path-smoke-r1/teacher_cache.jsonl
uv run edge-mvp run \
  --run-dir runs/l1-age-path-smoke-r1 \
  --teacher cache \
  --max-requests 40 \
  --compile-every 20
```

Result:

- Before this path, the comparable 40-request promotion-gate smoke had
  `L0=1`, `L1=1`, `L2=0`, `L3=0`, `L4=38`.
- After this path: `L0=1`, `L1=11`, `L2=0`, `L3=0`, `L4=28`.
- L1 accepted 11 requests and all 11 were frame-exact against teacher labels.
- The two promotion records both promoted with
  `promotion_block_layer_regressions=true`, `regressed_layers=[]`, and
  `promoted_with_layer_regression=false`.

## Conclusion

The path safely reduces L4 share on a repeated factoid-age family in the 40
request smoke. It is not yet a final quality result; next validation should run
the same code on 3k/10k cached replay after L2/L1 candidate changes accumulate.
