# Changelog

## v0.1.0-alpha.1 - Unreleased

This is the first public alpha preparation release.

Added:

- No-network `darjeeling demo thin-target` command.
- README narrative for first-time users.
- CI for lint, tests, and demo execution.
- GitHub issue forms for demo feedback and use cases.
- Alpha release notes and launch checklists.

Changed:

- PyPI distribution name is now `darjeeling-ai`.
- Python import package remains `darjeeling`.
- CLI command remains `darjeeling`.

Alpha limits:

- Stores and queues are filesystem or in-memory implementations.
- Current release is intended for design validation and early demo feedback.
- Production hardening remains future work.
